# Chapter 1: Founder Sales

In early-stage startups, founders should talk directly to customers.

```video
url: https://youtu.be/DH7REvnQ1y4?si=X53Y_PG5a2OpTG89
```

---

## 🧠 Key Idea

Founder-led sales helps you:
- Understand real customer pain
- Iterate faster
- Avoid building useless features

---
